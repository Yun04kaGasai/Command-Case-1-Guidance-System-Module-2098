# Описание модуля
## Модуль обеспечивает:
- Двухосевое наведение лазерного луча (±40° по каждой оси с шагом 10°), исходящего из геометрического центра корпуса CubeSat 1U;
- Приём управляющих команд по радиоканалу (nRF24L01);
- Последовательное выполнение четырёх тестовых сканирующих режимов: горизонтального, вертикального и двух диагональных;
- Автономную работу исполнительной части сразу после включения питания;
- Отправку телеметрической информации — включая уникальный ID устройства, текущие углы наведения и номер активного режима.
## Аппаратная часть включает:

- STM32 Blue Pill — главный контроллер (наземная станция), принимающий команду START от ПК и передающий последовательности позиций по радио;
- Raspberry Pi Pico (RP2040) — исполнительный модуль внутри CubeSat, управляющий двумя сервоприводами GH-S37A и лазером KY-008;
- Радиомодуль nRF24L01 для односторонней связи от STM32 к RP2040;
- Собственный 3D-напечатанный корпус, соответствующий стандарту CubeSat 1U, с учётом размещения лазера в геометрическом центре.
### Проект полностью реализован участниками: от проектирования механики и электронной схемы до разработки прошивки обеих микроконтроллеров и сборки макета.
